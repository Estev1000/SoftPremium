Web App de Remises “San Luis Rides”
Optimiza tu servicio de remises con una Web App moderna y lista para usar. “San Luis Rides” permite a tus clientes reservar viajes, conocer tus servicios y pagar fácilmente mediante Mercado Pago desde el celular o la computadora.

¿Qué es?
Web App responsive para remiseros/agencias, 24/7.
Construida con React y Tailwind CSS para una experiencia rápida, moderna y adaptable a móviles.
Marca actual: San Luis Rides (puede personalizarse con tu logo/nombre).

Reservas más simples: CTA “Reservar Ahora” destacado en la portada.
Pagos rápidos: flujo guiado para copiar el alias del remisero y abrir Mercado Pago en app o web.
Experiencia premium: diseño elegante, secciones claras y navegación con scroll suave.
Multi-conductor: lista de remiseros con nombre, alias y reputación visible.
Siempre disponible: interfaz optimizada para carga rápida y uso en móviles.

Cómo funciona el pago
El cliente elige un remisero y el monto.
La app copia el alias del remisero al portapapeles.
El cliente elige abrir Mercado Pago en la app (móvil) o en la web para completar el pago.
Nota: El alias de cada remisero es configurable en el código y se muestra junto al nombre y rating.

Preguntas frecuentes
¿Funciona en celular?
Sí, es responsive y detecta móvil para abrir la app de Mercado Pago cuando está disponible.
¿Puedo cambiar el nombre y logo?
Sí, totalmente personalizable.
¿Puedo agregar más remiseros?
Sí, se editan en la lista REMISEROS_LIST del archivo.
¿Necesito una cuenta de Mercado Pago?
Sí, cada remisero debe tener su alias propio para cobrar.
